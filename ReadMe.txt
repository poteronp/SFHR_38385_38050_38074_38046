Celem poprawnego uruchomienia aplikacji należy:
1. Posiadać zainstalowaną aplikację do Zarządzania Lokalną Bazą Danych - np. XAMPP
2. Utworzyć lokalną Bazę Danych o nazwie "projekt" i zaimportować dane z pliku projekt.sql

Poniższe kroki opisują uruchomienie aplikacji konkretnie pod aplikacją XAMPP w przypadku korzystania z innej aplikacji należy odtworzyć poniższe kroki analogicznie do systemu z którego się korzysta:
1. W lokalizacji C:\xampp\htdocs utworzyć folder AEH
2. Przekopiować do folderu całą zawartość z repozytorium GitHub
!WAŻNE - folder musi nazywać się dokładnie AEH, a także nie należy zmieniać nazw plików ani folderów oraz nie naruszać struktury, w przeciwnym wypadku mogą wystąpić błędy z funkcjonalnościami
3. Uruchomić przeglądarkę internetową, a następnie wpisać localhost/AEH/index.php

Aplikacja została uruchomiona - GRATULACJE
